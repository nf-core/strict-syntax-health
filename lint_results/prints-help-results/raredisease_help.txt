Input/output options
  --input                             [string] Path to the samplesheet file containing information about the samples in the experiment. 
  --outdir                            [string] The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  

Reference file options
  --bait_padding                      [integer] The amount to pad each end of the target intervals to create bait intervals. [default: 100] 
  --bwa                               [string]  Directory for pre-built bwa index. 
  --bwamem2                           [string]  Directory for pre-built bwamem2 index. 
  --bwameme                           [string]  Directory for pre-built bwameme's learned index. 
  --cadd_resources                    [string]  Path to the directory containing cadd annotations. 
  --fai                               [string]  Path to FASTA genome index file. 
  --fasta                             [string]  Path to FASTA genome file. 
  --gcnvcaller_model                  [string]  A file containing the path to models produced by GATK4 GermlineCNVCaller cohort. 
  --genome                            [string]  Name of iGenomes reference.  (accepted: GRCh37, GRCh38) [default: GRCh38] 
  --gnomad_af                         [string]  Path to the gnomad tab file with allele frequencies.

  --igenomes_ignore                   [boolean] Do not load the iGenomes reference config. 
  --intervals_wgs                     [string]  Path to the interval list of the genome (autosomes, sex chromosomes, and mitochondria).

  --known_dbsnp                       [string]  Path to known dbSNP file. 
  --known_dbsnp_tbi                   [string]  Path to known dbSNP file index.

  --ml_model                          [string]  Path to sentieon machine learning model file. 
  --mt_fasta                          [string]  Path to mitochondrial FASTA genome file. 
  --par_bed                           [string]  Path to a BED file containing PAR regions (used by deepvariant). 
  --ploidy_model                      [string]  Directory containing the ploidy model files 
  --readcount_intervals               [string]  Interval list file containing the intervals over which read counts are tabulated for CNV calling

  --sambamba_regions                  [string]  A BED file with regions of interest used in sambamba depth.

  --score_config_mt                   [string]  MT rank model config file for genmod. 
  --score_config_snv                  [string]  SNV rank model config file for genmod. 
  --score_config_sv                   [string]  SV rank model config file for genmod.

  --sequence_dictionary               [string]  Path to the genome dictionary file 
  --svdb_query_bedpedbs               [string]  Databases used for structural variant annotation in chrA-posA-chrB-posB-type-count-frequency format. 
 
  --svdb_query_dbs                    [string]  Databases used for structural variant annotation in vcf format. 
  --target_bed                        [string]  Path to directory for target bed file. 
  --variant_catalog                   [string]  Path to variant catalog file

  --verifybamid_svd_bed               [string]  Path to a BED file containing markers used by verifybamid2.

  --vep_filters                       [string]  Path to the file containing HGNC_IDs of interest on separate lines. 
  --vep_filters_scout_fmt             [string]  Path to a bed-like file exported by scout, which contains HGNC_IDs to be used in filter_vep. 
 

Analysis options
  --analysis_type                     [string]  Specifies which analysis type for the pipeline- either 'wgs', 'wes' or 'mito'. This changes 
resources consumed and tools used.  (accepted: wgs, wes, mito) [default: wgs]  
  --concatenate_snv_calls             [boolean] Specifies whether to generate a concatenated VCF file containing both nuclear & mitochondrial snv 
calls  
  --exclude_alt                       [boolean] After aligning the reads to a reference, remove alignments to alt contigs using samtools view, 
retaining only primary chromosomes (GRCh37: 1-22,X,Y,MT / GRCh38: chr1-chr22,chrX,chrY,chrM). 
 
  --extract_alignments                [boolean] After aligning the reads to a reference, extract alignments from specific regions/contigs and 
restrict the analysis to those regions/contigs.  
  --homoplasmy_af_threshold           [number]  Allele frequency threshold for mitochondrial variants. Variants with an AF above this threshold will 
be treated as homoplasmic and assigned a 1/1 genotype. Range 0-1.0. [default: 1.0]  
  --platform                          [string]  Specifies the platform on which the reads were sequenced. [default: illumina] 
  --ngsbits_samplegender_method       [string]  Method selection for ngs-bits samplegender  (accepted: xy, hetx, sry) [default: xy] 
  --restrict_to_contigs               [string]  Can be specified as RNAME[:STARTPOS[-ENDPOS]]. Multiple regions should be seperated by space 
 
  --run_mt_for_wes                    [boolean] Specifies whether to run mitochondrial analysis for wes samples 
  --run_rtgvcfeval                    [boolean] Specifies whether to run rtgtools' vcfeval 
  --run_vcfanno_db_sanity_check       [boolean] Specifies whether to check vcfanno database files for zero records and remove the corresponding 
annotation blocks from the TOML config before running vcfanno  
  --save_all_mapped_as_cram           [boolean] Specifies whether to generate and publish all (unfiltered) alignment files as cram instead of bam 
 
  --save_noalt_mapped_as_cram         [boolean] Specifies whether to generate and publish alt-filtered alignment files as cram instead of bam. 
Requires exclude_alt to be set to true.  
  --scatter_count                     [integer] Number of intervals to split your genome into (used to parallelize annotations) [default: 20] 
 
  --skip_subworkflows                 [string]  Disable specified subworkflows. 
  --skip_tools                        [string]  Disable specified tools. 

Alignment options
  --aligner                           [string]  Specifies the alignment algorithm to use - available options are 'bwamem2', 'bwa', 'bwameme' and 
'sentieon'.  (accepted: bwa, bwamem2, bwameme, sentieon) [default: bwamem2]  
  --mt_aligner                        [string]  Specifies the alignment algorithm to use - available options are 'bwamem2', 'bwa' and 'sentieon'. 
 (accepted: bwa, bwamem2, sentieon) [default: bwamem2]  
  --samtools_sort_threads             [integer] Number of threads allocated for sorting alignment files (used only by bwameme) [default: 4] 
  --mbuffer_mem                       [integer] Memory allocated for mbuffer in megabytes (used only by bwameme) [default: 3072] 
  --min_trimmed_length                [integer] Discard trimmed reads shorter than the given value [default: 40] 
  --mt_subsample_approach             [string]  Subsample mitochondria based on fraction of reads or number of reads  (accepted: fraction, 
reads) [default: reads]  
  --mt_subsample_reads                [integer] Expected number of reads to subsample mitochondria to, when mt_subsample_approach is set to reads 
[default: 18000]  
  --mt_subsample_rd                   [integer] Expected coverage to subsample mt alignment to, when mt_subsample_approach is set to fraction 
[default: 150]  
  --mt_subsample_seed                 [integer] Subsampling seed used to influence which subset of mitochondrial reads is kept. Used when 
mt_subsample_approach is set to fraction [default: 30]  
  --rmdup                             [boolean] Specifies whether duplicates reads should be removed prior to variant calling. 

Variant calling options
  --call_interval                     [string]  Interval in the reference that will be used in the software. Used only by sentieon. 
  --cnvnator_binsize                  [integer] Bin size for CNVnator [default: 1000] 
  --sentieon_dnascope_pcr_indel_model [string]  Option for selecting the PCR indel model used by Sentieon Dnascope.  (accepted: NONE, HOSTILE, 
AGGRESSIVE, CONSERVATIVE) [default: CONSERVATIVE]  
  --variant_caller                    [string]  Specifies the variant caller to use - available options are 'deepvariant' and 'sentieon'.  
(accepted: deepvariant, sentieon) [default: deepvariant]  
  --variant_type                      [string]  Specifies the variant types for sentieon variant caller.  (accepted: snp, indel, snp,indel) 
[default: snp,indel]  

Annotation options
  --variant_consequences_snv          [string]  File containing list of SO terms listed in the order of severity from most severe to lease severe 
for annotating genomic and mitochondrial SNVs.  
  --variant_consequences_sv           [string]  File containing list of SO terms listed in the order of severity from most severe to lease severe 
for annotating genomic SVs.  
  --vep_cache_version                 [integer] Specify the version of the VEP cache provided to the `--vep_cache` option. [default: 112] 

Mitosalt and saltshaker options
  --hisat2                            [string]  
  --hisat2_build_memory               [string]  Minimum memory required to build HISAT2 index with splice sites and exons. If available memory is 
below this threshold, a simpler index is built without splice sites.  
  --mitosalt_breakspan                [integer] minimum number of bases a non-split read must span either side of a breakpoint to be considered in 
the heteroplasmy count [default: 15]  
  --mitosalt_breakthreshold           [integer] the maximum deviation a given set of split read breakpoints can have to be considered within the 
same cluster [default: 2]  
  --mitosalt_cluster_threshold        [integer] minimum number of reads supporting a cluster [default: 5] 
  --mitosalt_deletion_threshold_max   [integer] the maximum size of the gap between fragments of a split read for the split read to be considered as 
potentially spanning a deletion [default: 30000]  
  --mitosalt_deletion_threshold_min   [integer] the minimum size of the gap between fragments of a split read for the split read to be considered as 
potentially spanning a deletion [default: 30]  
  --saltshaker_dominant_fraction      [number]  fraction for dominant fraction for saltshaker classification [default: 0.5] 
  --mitosalt_evalue_threshold         [number]  alignment e-value cutoff [default: 0.00001] 
  --mitosalt_exclude                  [integer] filter split reads if both fragments fall within the regions at the start and end to avoid 
pseudo-gapped alignments due to a circular genome [default: 5]  
  --mitosalt_flank                    [integer] basepairs flanking a deletion [default: 15] 
  --saltshaker_group_radius           [integer] spatial clustering radius for saltshaker grouping [default: 600] 
  --saltshaker_high_heteroplasmy      [integer] high heteroplasmy threshold for saltshaker classification [default: 10] 
  --mitosalt_heteroplasmy_limit       [number]  detected heteroplasmy threshold [default: 0.01] 
  --mitosalt_depth                    [integer] depth to subsample fastq files to before running mitosalt [default: 10000000] 
  --mito_length                       [integer] length of mitochondrial genome [default: 16569] 
  --saltshaker_multiple_threshold     [integer] threshold for multiple saltshaker classification [default: 5] 
  --saltshaker_noise_threshold        [number]  heteroplasmy threshold for background noise [default: 0.3] 
  --heavy_strand_origin_start         [integer] start of heavy strand origin [default: 16081] 
  --heavy_strand_origin_end           [integer] end of heavy strand origin [default: 407] 
  --light_strand_origin_start         [integer] start of light strand origin [default: 5730] 
  --light_strand_origin_end           [integer] end of light strand origin [default: 5763] 
  --mitosalt_paired_distance          [integer] the maximum distance of a paired read from its split counterpart for paired support to be considered 
positive [default: 1000]  
  --mitosalt_score_threshold          [integer] alignment score cutoff [default: 80] 
  --mitosalt_sizelimit                [integer] maximum size of deletions tolerated, beyond which they are reclassified as potential duplications in 
the reverse orientation [default: 10000]  
  --mitosalt_split_distance_threshold [integer] the maximum length of unmapped distance between two fragments of a split read [default: 5] 
  --mitosalt_split_length             [integer] minimum number of bases in a split read fragment for the read to be considered as potentially 
spanning a deletion [default: 15]
