Input/output options
  --input                            [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --outdir                           [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                            [string]  Email address for completion summary. 
  --multiqc_title                    [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified. 
  --dbname                           [string]  Specify name that resulting databases will be prefixed with. 
  --accession2taxid                  [string]  NCBI-style four-column accession to taxonomy ID map file 
  --prot2taxid                       [string]  Two column protein sequence accession ID to taxonomy map file. 
  --nucl2taxid                       [string]  Two column nucleotide sequence accession ID to taxonomy map file. 
  --nodesdmp                         [string]  Path to NCBI-style taxonomy node dmp file. 
  --namesdmp                         [string]  Path to NCBI-style taxonomy names dmp file. 
  --genomesizes                      [string]  Path to NCBI or GTDB genome sizes file 
  --malt_mapdb                       [string]  Path to MEGAN6/MALT mapping db file 
  --malt_mapdb_format                [string]  Specify the type of MALT mapdb provided, based on the corresponding MALT flag.  (accepted: mdb, 
a2t, s2t, a2ec, s2ec, t4ec, a2eggnog, s2eggnog, t4eggnog, a2gtdb, s2gtdb, t4gtdb, ...)  
  --save_concatenated_fastas         [boolean] Save concatenated input FASTAs 

Database Building Options
  --build_bracken                    [boolean] Turn on extending of Kraken2 database to include Bracken files. Requires nucleotide FASTA File 
input.  
  --bracken_build_options            [string]  Specify parameters being given to bracken build. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --build_centrifuge                 [boolean] Turn on building of Centrifuge database. Requires nucleotide FASTA file input. 
  --centrifuge_build_options         [string]  Specify parameters being given to centrifuge-build. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --build_diamond                    [boolean] Turn on building of DIAMOND database. Requires amino-acid FASTA file input. 
  --diamond_build_options            [string]  Specify parameters being given to diamond makedb. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --build_ganon                      [boolean] Turn on building of ganon database. Requires nucleotide FASTA file input. 
  --ganon_build_options              [string]  Specify parameters being given to ganon buildcustom. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --build_kaiju                      [boolean] Turn on building of Kaiju database. Requires amino-acid FASTA file input. 
  --kaiju_build_options              [string]  Specify parameters being given to kaiju-mkbwt. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --kaiju_keepintermediate           [boolean] Save intermediate files otherwise not required for downstream classification. 
  --build_kmcp                       [boolean] Turn on building of KMCP database. Requires nucleotide FASTA file input. 
  --kmcp_compute_options             [string]  Specify parameters being given to kmcp compute. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --kmcp_index_options               [string]  Specify parameters being given to kmcp index. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --build_kraken2                    [boolean] Turn on building of Kraken2 database. Requires nucleotide FASTA file input. 
  --kraken2_build_options            [string]  Specify parameters being given to kraken2 build. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --kraken2_keepintermediate         [boolean] Retain intermediate Kraken2 build files for inspection. 
  --build_krakenuniq                 [boolean] Turn on building of KrakenUniq database. Requires nucleotide FASTA file input. 
  --krakenuniq_build_options         [string]  Specify parameters being given to krakenuniq build. Parameters must be supplied in quotes: 
`--<tool>_build_options "--your_param"`.  
  --krakenuniq_keepintermediate      [boolean] Save intermediate files otherwise not required for downstream classification. 
  --build_malt                       [boolean] Turn on building of MALT database. Requires nucleotide FASTA file input.
