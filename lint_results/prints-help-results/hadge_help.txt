Input/output options
  --input                                   [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --mode                                    [string]  Mode of the pipeline.  (accepted: genetic, hashing, rescue, donor_match) [default: rescue] 
  --hash_tools                              [string]  Tools used for hash demultiplexing. [default: gmm-demux] 
  --genetic_tools                           [string]  Tools used for genetic demultiplexing. [default: vireo] 
  --bam_qc                                  [boolean] Perform BAM QC. [default: true] 
  --common_variants                         [string]  File with common variants. If provided, the BAM files will be filtered to only include reads that 
overlap with the common variants.  
  --outdir                                  [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --save_intermediates                      [boolean] Save intermediate files. 
  --email                                   [string]  Email address for completion summary. 
  --multiqc_title                           [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified.
