or `--helpFull`.  

Input/output options
  --input                         [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --outdir                        [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                         [string]  Email address for completion summary. 
  --multiqc_title                 [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified. 
  --save_intermediates            [boolean] Save all intermediate files (e.g. fastq, bams) of all steps of the pipeline  to output directory 
 

UMI options
  --with_umi                      [boolean] Enable UMI-based read deduplication. 
  --umitools_extract_method       [string]  UMI pattern to use. Can be either 'string' (default) or 'regex'. [default: string] 
  --umitools_method               [string]  UMI grouping method [default: dir] 
  --skip_umi_extract_before_dedup [boolean] Skip the UMI extraction from the reads before deduplication. Please note, if this parameter is set 
to false, the reads will be deduplicated solely on insert sequence. UMIs might be extracted after 
deduplication depending on the set umitools_bc_pattern nevertheless if with_umi is set to True. 
[default: true]  
  --umitools_bc_pattern           [string]  The UMI barcode pattern to use e.g. 'NNNNNN' indicates that the first 6 nucleotides of the read are 
from the UMI.  
  --umi_discard_read              [integer] After UMI barcode extraction discard either R1 or R2 by setting this parameter to 1 or 2, 
respectively.  
  --save_umi_intermeds            [boolean] If this option is specified, intermediate FastQ and BAM files produced by UMI-tools are also saved 
in the results directory.  

Reference genome options
  --genome                        [string]  Name of iGenomes reference. 
  --mirgenedb                     [boolean] Boolean whether MirGeneDB should be used instead of miRBase 
  --mirtrace_species              [string]  Species for miRTrace. 
  --mirgenedb_species             [string]  Species of MirGeneDB. 
  --fasta                         [string]  Path to FASTA genome file. 
  --mirna_gtf                     [string]  GFF/GTF file with coordinates positions of precursor and miRNAs. 
  --mirgenedb_gff                 [string]  GFF/GTF file with coordinates positions of precursor and miRNAs. 
  --mature                        [string]  Path to FASTA file with mature miRNAs. [default: 
https://github.com/nf-core/test-datasets/raw/smrnaseq/miRBase/mature.fa]  
  --mirgenedb_mature              [string]  Path to FASTA file with MirGeneDB mature miRNAs. 
  --hairpin                       [string]  Path to FASTA file with miRNAs precursors. [default: 
https://github.com/nf-core/test-datasets/raw/smrnaseq/miRBase/hairpin.fa]  
  --mirgenedb_hairpin             [string]  Path to FASTA file with miRNAs precursors. 
  --bowtie_index                  [string]  Path to a Bowtie 1 index directory 
  --save_reference                [boolean] Save generated reference genome files to results. 
  --igenomes_base                 [string]  Directory / URL base for iGenomes references. [default: s3://ngi-igenomes/igenomes/] 

Trimming options
  --clip_r1                       [integer] The number of basepairs to remove from the 5' end of read 1. 
  --three_prime_clip_r1           [integer] The number of basepairs to remove from the 3' end of read 1 AFTER adapter/quality trimming has been 
performed.  
  --three_prime_adapter           [string]  Sequencing adapter sequence to use for trimming. [default: AGATCGGAAGAGCACACGTCTGAACTCCAGTCA] 
 
  --trim_fastq                    [boolean] Trim FastQ files [default: true] 
  --fastp_min_length              [integer] Minimum filter length for raw reads. [default: 17] 
  --fastp_max_length              [integer] Maximum filter length for raw reads. [default: 100] 
  --save_trimmed_fail             [boolean] Save reads failing trimming 
  --fastp_known_mirna_adapters    [string]  Fasta with known miRNA adapter sequences for adapter trimming [default: 
${projectDir}/assets/known_adapters.fa]  
  --min_trimmed_reads             [integer] Minimum number of reads required in input file to use it [default: 10] 
  --save_merged                   [boolean] Save merged reads. 
  --phred_offset                  [integer] The PHRED quality offset to be used for any input fastq files. Default is 33, standard Illumina 1.8+ 
format. [default: 33]  

Contamination filter options
  --filter_contamination          [boolean] Enables the contamination filtering. 
  --rrna                          [string]  Path to the rRNA fasta file to be used as contamination database. 
  --trna                          [string]  Path to the tRNA fasta file to be used as contamination database. 
  --cdna                          [string]  Path to the cDNA fasta file to be used as contamination database. 
  --ncrna                         [string]  Path to the ncRNA fasta file to be used as contamination database. 
  --pirna                         [string]  Path to the piRNA fasta file to be used as contamination database. 
  --other_contamination           [string]  Path to an additional fasta file to be used as contamination database.

  --skip_multiqc                  [boolean] Skip MultiQC 
  --skip_fastp                    [boolean] Skip FastP
