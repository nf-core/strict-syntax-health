Input/output options
  --input                       [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --contrasts                   [string]  Path to comma-separated file containing information about the contrasts in the experiment. 
  --source                      [string]  Source of input files.  (accepted: fastq, genome_bam, transcriptome_bam, salmon_results) 
[default: fastq]  
  --outdir                      [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                       [string]  Email address for completion summary. 
  --multiqc_title               [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified. 
  --save_merged_fastq           [boolean] Save FastQ files after merging re-sequenced libraries in the results directory. 

Reference genome options
  --genome                      [string]  Name of iGenomes reference. 
  --fasta                       [string]  Path to FASTA genome file. 
  --gtf                         [string]  Path to GTF annotation file. 
  --gff                         [string]  Path to GFF3 annotation file. 
  --transcript_fasta            [string]  Path to FASTA transcriptome file. 
  --star_index                  [string]  Path to directory or tar.gz archive for pre-built STAR index. 
  --salmon_index                [string]  Path to directory or tar.gz archive for pre-built Salmon index. 
  --gencode                     [boolean] Specify if your transcript FASTA file is in GENCODE format. 
  --gtf_extra_attributes        [string]  By default, the pipeline uses the gene_name field to obtain additional gene identifiers from the 
input GTF file when running Salmon. [default: gene_name]  
  --gtf_group_features          [string]  Define the attribute type used to group features in the GTF file when running Salmon. [default: 
gene_id]  
  --save_reference              [boolean] If generated by the pipeline save the STAR index in the results directory. 

Generic options
  --multiqc_methods_description [string] Custom MultiQC yaml file containing HTML including a methods description. 

Read trimming options
  --clip_r1                     [integer] Instructs Trim Galore to remove bp from the 5' end of read 1 (or single-end reads). 
  --clip_r2                     [integer] Instructs Trim Galore to remove bp from the 5' end of read 2 (paired-end reads only). 
  --three_prime_clip_r1         [integer] Instructs Trim Galore to remove bp from the 3' end of read 1 AFTER adapter/quality trimming has been 
performed.  
  --three_prime_clip_r2         [integer] Instructs Trim Galore to remove bp from the 3' end of read 2 AFTER adapter/quality trimming has been 
performed.  
  --trim_nextseq                [integer] Instructs Trim Galore to apply the --nextseq=X option, to trim based on quality after removing 
poly-G tails.  
  --save_trimmed                [boolean] Save the trimmed FastQ files in the results directory. 
  --skip_trimming               [boolean] Use this if your input FastQ files have already been trimmed outside of the workflow or if you're 
very confident that there is no adapter contamination in your data.  
  --skip_trimgalore_fastqc      [boolean] Skip TrimGalore! FastQC. 
  --min_trimmed_reads           [integer] Minimum number of trimmed reads below which samples are flagged in multiqc output. [default: 
10000]  

QC options
  --skip_fastqc                 [boolean] Skip FastQC. 
  --skip_bigwig                 [boolean] Skip bigWig file creation. [default: true] 

Alignment options
  --aligner                     [string]  Specifies the alignment algorithm to use - available options are 'star_salmon', or 'star'.  
(accepted: star_salmon, star) [default: star_salmon]  
  --pseudo_aligner              [string]  Specifies the pseudo aligner to use - available options are 'salmon'. Runs in addition to 
'--aligner'.  (accepted: salmon) [default: salmon]  
  --bam_csi_index               [boolean] Create a CSI index for BAM files instead of the traditional BAI index. This will be required for 
genomes with larger chromosome sizes.  
  --star_ignore_sjdbgtf         [boolean] When using pre-built STAR indices do not re-extract and use splice junctions from the GTF file. 
 
  --salmon_quant_libtype        [string]   Override Salmon library type inferred based on strandedness defined in meta object. 
  --seq_center                  [string]  Sequencing center information to be added to read group of BAM files. 
  --skip_alignment              [boolean] Skip all of the alignment-based processes within the pipeline. 
  --save_unaligned              [boolean] Where possible, save unaligned reads from either STAR or Salmon to the results directory. 
  --save_align_intermeds        [boolean] Save the intermediate BAM files from the alignment step. 

Leafcutter options
  --leafcutter                  [boolean] Run Leafcutter annotation workflow.

  --rmats_splice_diff_cutoff    [number]  The cutoff used in the null hypothesis test for differential splicing. [default: 0.0001] 
  --rmats_paired_stats          [boolean] Use paired statistical model. [default: true] 
  --rmats_read_len              [integer] The length of each read. [default: 40] 
  --rmats_novel_splice_site     [boolean] Detect splicing events that involve an unannotated splice site. 
  --rmats_min_intron_len        [integer] Minimum Intron Length. [default: 50] 
  --rmats_max_exon_len          [integer] Maximum exon length. [default: 500]

SUPPA options
  --suppa                       [boolean] Run SUPPA workflow. [default: true] 
  --suppa_per_local_event       [boolean] Quantify event inclusion levels (PSIs) from multiple samples. [default: true] 
  --suppa_per_isoform           [boolean] Quantify isoform inclusion levels (PSIs) from multiple samples. [default: true] 
  --suppa_tpm                   [string]  Expression file containing the abundances of all transcripts (ideally in TPM units). 
  --generateevents_pool_genes   [boolean] Redefine genes by clustering together transcripts by genomic stranded overlap and sharing at least 
one exon. [default: true]  
  --generateevents_event_type   [string]  Space separated list of events to generate. [default: SE SS MX RI FL] 
  --generateevents_boundary     [string]  Boundary type (only used for local AS events).  (accepted: S, V) [default: S] 
  --generateevents_threshold    [integer] Variability treshold. [default: 10] 
  --generateevents_exon_length  [integer] Defines the number of nucleotides to display in the output GTF. [default: 100] 
  --psiperevent_total_filter    [integer] Minimum total expression of the transcripts involved in the event. [default: 0] 
  --diffsplice_local_event      [boolean] Calculate differential splicing for AS events across multiple conditions with replicates. 
[default: true]  
  --diffsplice_isoform          [boolean] Calculate differential splicing for differential transcript usage across multiple conditions with 
replicates [default: true]  
  --diffsplice_method           [string]  The method to use to calculate the significance.  (accepted: empirical, classical) [default: 
empirical]  
  --diffsplice_area             [integer] Integer indicating the number of points in the local area of the delta PSI - average TPM 
distribution. [default: 1000]  
  --diffsplice_lower_bound      [integer] Lower-bound for the absolute delta PSI value to test for significance. [default: 0] 
  --diffsplice_gene_correction  [boolean] Correct the p-values by gene. [default: true] 
  --diffsplice_paired           [boolean] Indicates if replicates across conditions are paired. [default: true] 
  --diffsplice_alpha            [number]  Family-wise error rate to use for the multiple test correction. [default: 0.05] 
  --diffsplice_median           [boolean] Use the median to calculate the Delta PSI, instead of the mean. 
  --diffsplice_tpm_threshold    [integer] Minimum expression (calculated as average TPM value within-replicates and between-conditions) to be 
included in the analysis. [default: 0]  
  --diffsplice_nan_threshold    [integer] Proportion of samples with nan values allowed per condition to calculate a DeltaPSI . [default: 
0]  
  --clusterevents_local_event   [boolean] Cluster events according to PSI values across conditions [default: true] 
  --clusterevents_isoform       [boolean] Cluster transcripts according to PSI values across conditions [default: true] 
  --clusterevents_sigthreshold  [number]  P-value threshold to consider an event significant from the dpsi file. 
  --clusterevents_dpsithreshold [number]  Lower-bound for the absolute delta PSI value to cluster. [default: 0.05] 
  --clusterevents_eps           [number]  Maximum distance (between 0 and 1) to consider two events as members of the same cluster. 
[default: 0.05]  
  --clusterevents_metric        [string]  Distance metric.  (accepted: euclidean, manhattan, cosine) [default: euclidean] 
  --clusterevents_separation    [integer] Maximum distance in PSI space of an event to a cluster. 
  --clusterevents_min_pts       [integer] Minimum number of events required per cluster. [default: 20] 
  --clusterevents_method        [string]  Clustering method to use (DBSCAN, OPTICS).  (accepted: OPTICS, DBSCAN) [default: DBSCAN]
