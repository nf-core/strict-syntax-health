Input/output options
  --input                       [string]  Path to comma-separated file containing information about the samples in the experiment. 
  --outdir                      [string]  The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --primers                     [string]  Fasta file of primers sequences 
  --gtf                         [string]  Genome annotation file 
  --chunk_ccs                   [integer] Defines the number of batches to run in parallel for. [default: 40] 
  --chunk_mapping               [integer] Splits fasta in files of 'chunk' sequences. [default: 100] 
  --email                       [string]  Email address for completion summary. 
  --multiqc_title               [string]  MultiQC report title. Printed as page header, used for filename if not otherwise specified. 

CCS options
  --rq                          [number]  ccs --rq option, define the minimum read quality for CCS selection [default: 0.9] 
  --min_passes                  [integer] ccs --min-passes option, define the minimum number of passes to select a CCS [default: 3] 
  --min_snr                     [number]  ccs --min-snr option, minimum SNR of subreads to use for generating CCS [default: 2.5] 
  --min_length                  [integer] ccs --min-length option, minimum CCS length for CCS selection [default: 10] 
  --max_length                  [integer] ccs --max-length option, maximum CCS length for CCS selection [default: 50000] 
  --top_passes                  [integer] ccs --top-passes option, maximum number of passes to use for CCS generation [default: 60] 

Aligner option
  --aligner                     [string] Aligner to use for mapping: minimap2 or ultra  (accepted: minimap2, ultra) 

TAMA options
  --capped                      [boolean] TAMA collapse: Capped RNA? 
  --five_prime                  [integer] TAMA collapse: 5 prime wobble threshold [default: 100] 
  --splice_junction             [integer] TAMA collapse: Splice junction / exon wobble threshold [default: 10] 
  --three_prime                 [integer] TAMA collapse: 3 prime wobble threshold [default: 100] 
  --tama_merge_all              [boolean] TAMA merge: merge the sample-wise annotations into a single annotation? [default: false] 

Reference genome options
  --genome                      [string] Name of iGenomes reference. 
  --fasta                       [string] Path to FASTA genome file. 

Generic options
  --multiqc_methods_description [string]          Custom MultiQC yaml file containing HTML including a methods description.
