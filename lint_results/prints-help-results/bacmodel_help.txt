Input/output options
  --input                 [string] Path to comma-separated or tab-separated file containing information about the samples in the 
experiment.

Cloud infrastructure.  
  --email                 [string] Email address for completion summary. 

Generic options
  --help                  [boolean, string] Display the help message. 
  --help_full             [boolean]         Display the full detailed help message. 
  --show_hidden           [boolean]         Display hidden parameters in the help message (only works when --help or --help_full are provided). 
 

Annotation options
  --annotation_tool       [string]  Tool to use for genome annotation  (accepted: prokka, bakta) [default: prokka] 
  --baktadb               [string]  Path to Bakta database (required if annotation_tool='bakta') 
  --baktadb_download      [boolean] Download Bakta database automatically 
  --baktadb_download_args [string]  Arguments for Bakta database download [default: --type light] 

Database options
  --pfamdb                [string]  Path to Pfam database (required if run_traitar=true) 
  --pfamdb_download       [boolean] Download Pfam database automatically 

Analysis options
  --run_macsyfinder       [boolean] Run MacSyFinder for macromolecular systems detection [default: true]

  --run_traitar           [boolean] Run TRAITAR for phenotype prediction [default: true] 
  --run_carveme           [boolean] Run CarveMe for genome-scale metabolic reconstruction [default: true] 
  --carveme_mediadb       [string]  Path to CarveMe media database TSV file. If not provided, uses CarveMe's default media database
