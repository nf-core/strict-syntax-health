Typical pipeline command:

  nextflow run nf-core/sarek -profile <docker/singularity/.../institute> --input samplesheet.csv --outdir <OUTDIR>

help message of that parameter will be printed.  
or `--helpFull`.  

Input/output options
  --input                             [string] Path to comma-separated file containing information about the samples in the experiment. 
  --step                              [string] Starting step  (accepted: mapping, markduplicates, prepare_recalibration, recalibrate, 
variant_calling, annotate) [default: mapping]  
  --outdir                            [string] The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  

Main options
  --split_fastq                       [integer] Specify how many reads each split of a FastQ file contains. Set 0 to turn off splitting at all. 
[default: 50000000]  
  --nucleotides_per_second            [integer] Estimate interval size. [default: 200000] 
  --intervals                         [string]  Path to target bed file in case of whole exome or targeted sequencing or intervals file. 
  --no_intervals                      [boolean] Disable usage of intervals. 
  --wes                               [boolean] Enable when exome or panel data is provided. 
  --tools                             [string]  Tools to use for contamination removal, duplicate marking, variant calling and/or for annotation. 
 
  --skip_tools                        [string]  Disable specified tools. 

FASTQ Preprocessing
  --trim_fastq                        [boolean] Run FastP for read trimming 
  --clip_r1                           [integer] Remove bp from the 5' end of read 1 [default: 0] 
  --clip_r2                           [integer] Remove bp from the 5' end of read 2 [default: 0] 
  --three_prime_clip_r1               [integer] Remove bp from the 3' end of read 1 [default: 0] 
  --three_prime_clip_r2               [integer] Remove bp from the 3' end of read 2 [default: 0] 
  --trim_nextseq                      [boolean] Removing poly-G tails. 
  --length_required                   [integer] Minimum length of reads to keep [default: 15] 
  --save_trimmed                      [boolean] Save trimmed FastQ file intermediates. 
  --save_split_fastqs                 [boolean] If set, publishes split FASTQ files. Intended for testing purposes. 

Unique Molecular Identifiers
  --umi_read_structure                [string]  Specify UMI read structure for fgbio UMI consensus read generation 
  --group_by_umi_strategy             [string]  Default strategy for fgbio UMI-based consensus read generation   (accepted: Identity, Edit, 
Adjacency, Paired) [default: Adjacency]  
  --umi_in_read_header                [boolean] Move UMIs from fastq read headers to a tag prior to deduplication. 
  --umi_location                      [string]  Location of the UMI(s) to be extracted with fastp.  (accepted: read1, read2, per_read, index1, 
index2, per_index)  
  --umi_length                        [integer] Length of the UMI(s) in the read. 
  --umi_base_skip                     [integer] Number of bases to skip after the UMI(s) in the read when extracting with fastp. 
  --umi_tag                           [string]  Tag detailing where UMIs are present inside the bam/cram file (e.g. RX). 
  --bbsplit_fasta_list                [string]  Path to comma-separated file containing a list of reference genomes to filter reads against with 
BBSplit. You have to also explicitly set `--tools bbsplit` if you want to use BBSplit.  
  --bbsplit_index                     [string]  Path to directory or tar.gz archive for pre-built BBSplit index. 
  --save_bbsplit_reads                [boolean] If this option is specified, FastQ files split by reference will be saved in the results directory. 
 

Preprocessing
  --aligner                           [string]  Specify aligner to be used to map reads to reference genome.  (accepted: bwa-mem, bwa-mem2, 
dragmap, sentieon-bwamem, parabricks) [default: bwa-mem]  
  --save_mapped                       [boolean] Save mapped files. 
  --save_output_as_bam                [boolean] Saves output from mapping (if `--save_mapped`), Markduplicates & Baserecalibration as BAM file 
instead of CRAM  
  --use_gatk_spark                    [string]  Enable usage of GATK Spark implementation for duplicate marking and/or base quality score 
recalibration  
  --markduplicates_pixel_distance     [integer] 
  --sentieon_consensus                [boolean] Generate consensus reads with Sentieon dedup rather than choosing one best read. 

Variant Calling
  --only_paired_variant_calling       [boolean] If true, skips germline variant calling for matched normal to tumor sample. Normal samples without 
matched tumor will still be processed through germline variant calling tools.  
  --ascat_min_base_qual               [integer] Overwrite Ascat min base quality required for a read to be counted. [default: 20] 
  --ascat_min_counts                  [integer] Overwrite Ascat minimum depth required in the normal for a SNP to be considered. [default: 10] 
 
  --ascat_min_map_qual                [integer] Overwrite Ascat min mapping quality required for a read to be counted. [default: 35] 
  --ascat_ploidy                      [number]  Overwrite ASCAT ploidy. 
  --ascat_purity                      [number]  Overwrite ASCAT purity. 
  --cf_chrom_len                      [string]  Specify a custom chromosome length file. 
  --cf_coeff                          [number]  Overwrite Control-FREEC coefficientOfVariation [default: 0.05] 
  --cf_contamination_adjustment       [boolean] Overwrite Control-FREEC contaminationAdjustement 
  --cf_contamination                  [integer] Design known contamination value for Control-FREEC [default: 0] 
  --cf_minqual                        [integer] Minimal sequencing quality for a position to be considered in BAF analysis. [default: 0] 
  --cf_mincov                         [integer] Minimal read coverage for a position to be considered in BAF analysis. [default: 0] 
  --cf_ploidy                         [string]  Genome ploidy used by ControlFREEC [default: 2] 
  --cf_window                         [number]  Overwrite Control-FREEC window size. 
  --cnvkit_reference                  [string]  Copy-number reference for CNVkit 
  --freebayes_filter                  [string]  Filtering expression for vcflib/vcffilter [default: 30] 
  --joint_germline                    [boolean] Turn on the joint germline variant calling for GATK haplotypecaller 
  --joint_mutect2                     [boolean] Runs Mutect2 in joint (multi-sample) mode for better concordance among variant calls of tumor 
samples from the same patient. Mutect2 outputs will be stored in a subfolder named with patient ID 
under `variant_calling/mutect2/` folder. Only a single normal sample per patient is allowed. 
Tumor-only mode is also supported.  
  --ignore_soft_clipped_bases         [boolean] Do not analyze soft clipped bases in the reads for GATK Mutect2. 
  --pon                               [string]  Panel-of-normals VCF (bgzipped) for GATK Mutect2 
  --pon_tbi                           [string]  Index of PON panel-of-normals VCF. 
  --sentieon_haplotyper_emit_mode     [string]  Option for selecting output and emit-mode of Sentieon's Haplotyper. [default: variant] 
  --sentieon_dnascope_emit_mode       [string]  Option for selecting output and emit-mode of Sentieon's Dnascope. [default: variant] 
  --sentieon_dnascope_pcr_indel_model [string]  Option for selecting the PCR indel model used by Sentieon Dnascope. [default: CONSERVATIVE] 
  --gatk_pcr_indel_model              [string]  [default: CONSERVATIVE]
