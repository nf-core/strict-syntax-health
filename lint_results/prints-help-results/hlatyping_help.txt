Input/output options

  --tools                       [string] Specifies the hlatyping tool(s) to use. [default: optitype]

  --solver                      [string]  Specifies the integer programming solver. [default: glpk] 
  --enumerations                [integer] Specifies the number of output solutions. [default: 1]
