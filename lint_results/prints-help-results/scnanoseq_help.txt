Input/output options
  --input                               [string] Path to comma-separated file containing information about the samples in the experiment. 
  --outdir                              [string] The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                               [string] Email address for completion summary. 
  --multiqc_title                       [string] MultiQC report title. Printed as page header, used for filename if not otherwise specified. 

Reference genome options
  --genome_fasta                        [string] Path to genome FASTA file. 
  --transcript_fasta                    [string] Path to transcriptome FASTA file. 
  --gtf                                 [string] Path to GTF file. 
  --genome                              [string] Name of iGenomes reference. 
  --fasta_delimiter                     [string] This is the delimiter the FASTA uses in the sequence identifier to separate the string. 

Fastq options
  --split_amount                        [integer] The number of lines to split  the FASTQ into. [default: 0] 

Read trimming options
  --min_length                          [integer] Choose minimum read length. [default: 1] 
  --min_q_score                         [integer] Choose minimum average read quality score. [default: 10] 
  --skip_trimming                       [boolean] Skip quality trimming step. 

Cell barcode options
  --whitelist                           [string]  User-provided file containing a list of cellular barcodes. Using this parameter will override the 
default whitelists provided by the pipeline and use the user-provided one instead.  
  --barcode_format                      [string]  Specify the format for the barcode+umi. This parameter also defines a default barcode whitelist for 
the pipeline to use for barcode calling, this can be overridden with the 'whitelist' parameter.  
(accepted: 10X_3v3, 10X_3v4, 10X_5v2, 10X_5v3)  
  --skip_blaze_demux                    [boolean] Do not perform the demultiplexing step in BLAZE (conserves time and disk space). [default: true] 
 
  --dedup_tool                          [string]  Specify which tool to be used for deduplication (Options: picard, umitools)  (accepted: 
umitools, picard) [default: umitools]  

Mapping
  --stranded                            [string]  Library strandness option.  (accepted: None, reverse, forward) 
  --kmer_size                           [integer] Minimizer k-mer length. [default: 14] 
  --save_genome_secondary_alignment     [boolean] Save the secondary alignments when aligning to the genome 
  --save_transcript_secondary_alignment [boolean] Save the secondary alignments when aligning to the transcriptome [default: true] 

Analysis options
  --retain_introns                      [boolean] Indicate whether to include introns in the count matrices [default: true] 
  --quantifier                          [string]  Provide a comma-delimited options of quantifiers for the pipeline to use. Available tools: isoquant, 
oarfish  

Process skipping options
  --skip_qc                             [boolean] Skip all QC. 
  --skip_fastqc                         [boolean] Skip FastQC. 
  --skip_nanoplot                       [boolean] Skip Nanoplot. 
  --skip_toulligqc                      [boolean] Skip ToulligQC. 
  --skip_fastq_nanocomp                 [boolean] Skip NanoComp from FASTQ file(s). 
  --skip_bam_nanocomp                   [boolean] Skip NanoComp from BAM file(s). [default: true] 
  --skip_rseqc                          [boolean] Skip RSeQC. 
  --skip_seurat                         [boolean] Skip Seurat QC. 
  --skip_save_minimap2_index            [boolean] Skip saving minimap2 index. 
  --skip_dedup                          [boolean] Skip umi dedup. 
  --skip_multiqc                        [boolean] Skip MultiQC. 

Generic options
