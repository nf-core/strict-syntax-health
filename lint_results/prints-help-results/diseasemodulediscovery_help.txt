Typical pipeline command:

help message of that parameter will be printed.  
or `--helpFull`.  

Input/output options
  --input                       [string] Path to a CSV sample sheet defining seed file-network combinations 
  --seeds                       [string] Path(s) to one or multiple file(s) with seed genes 
  --network                     [string] Path(s) to one or multiple PPI network(s) in gt, csv, tsv, graphml, xml, dot, or gml format. 
 
  --id_space                    [string] Type of gene/protein ids  (accepted: entrez, symbol, ensembl, uniprot) [default: entrez] 
  --outdir                      [string] The output directory where the results will be saved. You have to use absolute paths to storage on 
Cloud infrastructure.  
  --email                       [string] Email address for completion summary. 
  --multiqc_title               [string] MultiQC report title. Printed as page header, used for filename if not otherwise specified. 

Network expansion
  --skip_firstneighbor          [boolean] Flag for skipping first neighbor 
  --skip_domino                 [boolean] Flag for skipping DOMINO 
  --skip_robust                 [boolean] Flag for skipping Robust 
  --skip_robust_bias_aware      [boolean] Flag for skipping Robust (bias-aware) 
  --skip_diamond                [boolean] Flag for skipping DIAMOnD 
  --diamond_n                   [integer] Desired number of DIAMOnD genes. [default: 200] 
  --diamond_alpha               [integer] Weight of the seeds. [default: 1] 
  --skip_rwr                    [boolean] Flag for skipping random walk with restart 
  --rwr_scaling                 [boolean] Add a scaling depending on the node's degree. 
  --rwr_symmetrical             [boolean] Compute the symmetric instead of column-wise normalized Markov matrix. 
  --rwr_r                       [number]  Damping factor/restart probability. [default: 0.8] 

Module visualization
  --skip_visualization          [boolean] Skip module visualization 
  --visualization_max_nodes     [integer] If a module has more nodes it will not be visualized. [default: 500] 
  --skip_drugstone_export       [boolean] Flag for skipping the export to the Drugst.One platform 
  --drugstone_max_nodes         [integer] If a module has more nodes it will not be exported to the Drugst.One platform. Includes drug 
prioritization. [default: 500]  

Module annotation
  --skip_annotation             [boolean] Flag for skipping the annotation part of the process. 
  --validate_online             [boolean] Flag for validating online at baderlab.org. 
  --add_variants                [boolean] Flag for adding variants to the biopax annotation file. 

Module evaluation
  --skip_evaluation             [boolean] Flag skipping the entire evaluation workflow 
  --skip_gprofiler              [boolean] Flag for skipping g:Profiler 
  --skip_digest                 [boolean] Flag for skipping DIGEST 
  --skip_digest_reference_free  [boolean] Flag for skipping only the reference-free mode of DIGEST 
  --skip_digest_reference_based [boolean] Flag for skipping only the reference-based mode of DIGEST 
  --run_seed_perturbation       [boolean] Flag for running the seed perturbation-based evaluation 
  --run_network_perturbation    [boolean] Flag for running the network perturbation-base evaluation 
  --n_network_perturbations     [integer] Number of times the network will be perturbed for the network perturbation-based evaluation 
[default: 100]  
  --perturbed_networks          [string]  Path(s) to folder(s) with pre-computed perturbed networks for the network perturbation-based 
evaluation  

Drug prioritization
  --run_proximity               [boolean] Flag for running proximity 
  --skip_drug_predictions       [boolean] Flag for skipping drug predictions 
  --includeIndirectDrugs        [boolean] Drugst.One parameter for including indirect drugs. 
  --includeNonApprovedDrugs     [boolean] Drugst.One parameter for including non approved drugs. 
  --result_size                 [integer] Drugst.One parameter for defining  the maximum number of returned drugs. [default: 50] 
  --drugstone_algorithms        [string]  Drugst.One parameter for algorithms to be used. Comma separated list. Options: 'trustrank', 'degree' 
and 'closeness'. [default: trustrank]
